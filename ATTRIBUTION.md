## Blocked source class

FableLayer intentionally excludes private, proprietary, or non-public prompt text. Repositories or documents that appear to redistribute non-public system prompts are treated as **blocked/reference-only**. They are not named as dependencies, are not copied, and are not required to use FableLayer.

## Policy

- `copy` or `adapt` requires a verified compatible license.
- `reference-only`, `blocked`, and `unverified` sources must not be copied into runtime bundles.
- Performance claims require benchmark raw data and a limitations section.
- Generated runtime profiles must state that no private prompt text is included.
