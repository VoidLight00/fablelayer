# FableLayer Legal/Safety Review

## 현재 판정

- AGPL-3.0 목표는 가능하지만 참조 저장소별 LICENSE 확인 전 코드를 복사하지 않는다.
- 유출 시스템 프롬프트로 보이는 자료는 `reference-only`로만 취급한다.
- FableLayer 산출물은 “비공개 프롬프트 복제”가 아니라 “공개-safe 절차 discipline”으로 포지셔닝한다.

## Release Blockers

1. 각 GitHub reference의 LICENSE 확인 필요.
2. README에 유출 prompt 비포함 정책 명시 필요.
3. 패키지 publish 전 사용자 승인 필요.
