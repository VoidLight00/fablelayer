# Security Policy

FableLayer is local-first and does not require secrets for normal use.

## Reporting

Open a private security advisory or contact the maintainer before publishing details for:

- secret leakage
- unsafe source redistribution
- prompt text that appears proprietary or non-public
- gate bypasses that allow unverified performance or publish claims

## Public-safe source policy

FableLayer does not include private or proprietary system prompt text. High-risk prompt-source references are treated as blocked/reference-only and are not copied into runtime bundles.
